jQuery.safetynet

Example

An out-bound link

Steps to demonstrate behavior

  1. Try clicking on link or refreshing, no warning should occur
  2. Try modifing an input and select and navigating away. A warning should occur
  3. Change select back to its original value, "b", and try navigating away. A warning should still occur
  4. Change all inputs back to their original values and try navigating away. No warning should occur.
  5. Try changing values from their original values and clicking "Submit". No warning should occur
